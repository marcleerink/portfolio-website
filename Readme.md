## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/marcleerink/portfolio-website.git
   ```
2. Create virtual environment
   Linux/MacOS:
   ```sh
   python3 -m venv venv
   ```
   Powershell:
   ```sh
   py -3 -m venv venv
   ```
3. Activate virtual environment
   Linux/MacOS:
   ```sh
   venv/bin/activate
   ```
   Powershell:
   ```sh
   venv\Scripts\activate
   ```
4. Install dependencies
   ```sh
   pip install -r requirements.txt
   ```
5. Run project
   ```sh
   flask run
   ```
